name: figma
description: Read Figma designs and export assets using MCP tools
---

Figma is available via MCP tools. `FIGMA_ACCESS_TOKEN` is pre-configured.

Common operations available via MCP:
- Get file structure and components
- Read design properties (colors, fonts, dimensions)
- Export assets (images, icons)
- List projects and files

When working with Figma, always get the file key from the URL: `figma.com/design/<file-key>/...`
