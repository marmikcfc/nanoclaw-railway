---
name: google-drive
description: Read and write files in Google Drive using MCP tools
---

Google Drive is available via MCP tools. Use the provided MCP tools to list, read, and write files.

Common operations available via MCP:
- List files in a folder
- Read file contents
- Create and update files
- Search for files by name or content

`GOOGLE_CREDENTIALS` contains the service account JSON for authentication.
