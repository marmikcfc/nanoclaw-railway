---
name: agentmail
description: Send and receive emails as the agent using AgentMail MCP tools
---

Email is available via AgentMail MCP tools. `AGENTMAIL_API_KEY` is pre-configured.

Common operations available via MCP:
- Send email to one or more recipients
- Read inbox messages
- Reply to threads
- Search emails

When you receive a webhook event from AgentMail, it means a new email has arrived — read and respond to it.
